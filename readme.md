## 🚀 快速开始

### 📲 安装应用
1. 前往 [Releases](./releases) 下载最新版本
2. 安装APK文件到Android设备

### 🔌 安装插件

#### ikun音源插件（推荐）
```
https://musicfree.3kddyys.workers.dev/api/subscription.json?key=你的卡密.json
```
> 💡 提示：导入失败请使用代理，导入不完全请多次重试

#### 插件安装步骤
1. 打开应用 → 侧边栏 → 设置 → 插件设置
2. 选择"从网络安装插件"
3. 输入插件地址并确认
